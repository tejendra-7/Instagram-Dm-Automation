# Privacy Policy

Last updated: 2026

This application is operated by Tejendra for the purpose of automating responses on Instagram using the Instagram Graph API.

## Information We Collect

This application may process the following information through Instagram webhooks:

- Instagram user ID
- Instagram comments
- Instagram message events

This information is used only to trigger automated responses configured by the application owner.

## How We Use Information

The collected information is used solely for:

- Detecting keywords in Instagram comments
- Sending automated direct messages
- Providing requested links or information

No personal data is stored permanently or sold to third parties.

## Data Storage

This application does not store personal user data in a database. Webhook events are processed in real time and discarded after processing.

## Third-Party Services

This application uses the following services:

- Meta Instagram Graph API
- Vercel serverless hosting

These services may process data according to their own privacy policies.
